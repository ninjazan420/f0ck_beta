# f0ck.org Beta v1

Modern, feature-rich imageboard platform built with Next.js 14 and TailwindCSS.

## ⚡ Core Features

### Content
- 7x4 Grid Layout
- Smart Previews
- Safety Ratings
- Media Types Support
- Comment System
- User Profiles

### Premium 💎
- Custom Nicknames
- Advanced Stats
- Priority Support

## 🛠️ Quick Start

```bash
git clone https://github.com/f0ck-org/f0ck.git
cd f0ck
npm install
npm run dev
```

## 🔧 Tech Stack
- Next.js 14
- TailwindCSS
- Geist Font Family
- React Components
- Vercel Deployment
