### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation

1. Clone repository
```
git clone https://github.com/f0ck-org/f0ck.git
```

2. Change to project directory
```
cd f0ck
```

3. Install dependencies
```
npm install
```
or
```
yarn install
```

4. Start development server
```
npm run dev
```
or
```
yarn dev
```

5. Open in browser
```
http://localhost:3000
```
